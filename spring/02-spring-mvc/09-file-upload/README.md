# File Upload

## 학습 목표

File Upload의 핵심 개념을 이해하고
Spring에서 동작하고 활용되는 방식을 학습합니다.

## 주요 학습 내용

- File Upload
- 핵심 동작 원리
- Spring에서의 적용 방식
- 실무 사용 사례와 주의점

## 작성 방법

1.

루트 폴더의

`spring/_template.md`

를 복사합니다.

2.

파일명을 자신의 이름 또는 GitHub ID로 변경합니다.

예시

`jaemin.md`

`subin.md`

`minsu.md`

3.

자신의 파일 하나만 계속 수정합니다.

질문은 같은 파일 안에 계속 이어서 작성합니다.

새로운 질문마다 파일을 만들지 않습니다.

4.

다른 스터디원의 파일은 수정하지 않습니다.

## 작성 규칙

- 한 사람당 파일 하나만 사용합니다.
- 질문은 하나의 파일 안에 계속 추가합니다.
- 답변은 실제 면접에서 30초~1분 정도로 설명할 수 있는 분량으로 작성합니다.
- 정의뿐 아니라 이유와 동작 원리도 함께 작성합니다.
- 주요 질문에는 꼬리질문을 함께 작성합니다.
- 프로젝트 경험과 연결될 수 있는 내용은 함께 기록합니다.
- 참고 자료가 있다면 문서 하단에 작성합니다.
